# Abhinav Portfolio (React + Vite)

Files included:
- index.html (Vite entry)
- src/ (React source)
- package.json (scripts: dev, build, preview)
- tailwind.config.cjs, postcss.config.cjs

Quick start:
1. `cd react-portfolio`
2. `npm install`
3. `npm run dev`

Notes:
- Install Node 18+
- Tailwind is configured via PostCSS. You can adapt to your CI/CD or Cloudflare Pages.
